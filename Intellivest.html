Program Description

The initial phase of the program is to predict the user's credit score; this is done through a series of prompts whose responses are used to train a Random Forest Ensemble model that will predict the score

With the predicted score, the user will produce a series of questions for the user to determine whether they wish to invest, how much cash they'd be willing to input, and how large of a loan, if any at all, they're willing to take for their investment. Based on their credit score and the answers to the said prompts, the user is given certain investment advice

Technologies Used

  1. sklearn Library
  2. Random Forest Ensembles
  3. Numpy & Pandas
  4. Objects
  5. tkinter Desktop Output library
