Welcome to... the Space Race! Not the Soviet-American one, but the JavaScript one

This is a fun two-player game that I made in JavaScript where each user is trying to make it to the opposite side of the screen to beat the other player while doing their best to dodge the incoming asteroids (so maybe it IS a little like the Cold War one)

Technologies Used

  2. p5.js library
