Program Description

This program will comprehend the user's mental proccess for task prioritization by generating random events with different levels of criteon (difficulty, academic factor, time proximity, .etc)

A Random Forest Ensemble, using a Recursive Feature Elimination (RFE) model, will be trained with the results of this prompting and will then organize the user's ACTUAL to-do list. The program will iterate through the list and remind the user of the next event every 'n' minutes (determined by user)

Technologies Used

  1. sklearn Library
  2. Random Forest Ensembles
  3. Recursive Feature Elimination
  4. Numpy & Pandas
  5. Objects
  6. plyer Notification library
