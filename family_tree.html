Program Description

Reading in data from a 2d Array , the program counstructs a "Family Tree" Linked List that keeps generational relationships intact. The user can then acquire a list of any person's relatives

While a Binary/RB tree seems more intuitive, the program required two seperate pointers and didn't need to be sorted

The average time complexity is O(N)

Technologies Used

  1. Linked Lists
  2. Hashsets
  3. 2D Arrays
  4. Recursion
