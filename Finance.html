Program Description

This is Theta Terminal. It has various features which all essentially serve to suggest trading for different stocks. This includes means of research, prediction, and visualization. I plan to automate this proccess once I integrate most of the ideas I have to it

This project is a work in progress, as I'm always getting new ideas and trying to program them. The version you will see in the Github is simply a version of the project, and there will be no "fluid" input/UI proccess

Technologies Used

  1. Yahoo! Finance API
  2. JSON API Queries
  3. matplotlib Graphing
  4. Random Forest Ensembles
  5. Numpy & Pandas
