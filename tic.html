Program Description

This is the usual Tic-Tac-Toe, but with a twist: only 'n' amount of x/o objects are allowed on the board at once, the user gets to decide how many pieces need to be in a row to win, and the user determines the dimensions of the board.

The average time complexity is O(N), only N^2 when printing

Technologies Used

  1. Linked Lists (for storing peices)
  2. Hashsets (for tracking positions)
  3. Input Scanner
  4. Objects
